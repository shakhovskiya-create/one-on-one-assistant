## Задача

Исправить дизайн страницы Задачи (/tasks) для соответствия макету 01-tasks.html.

## Проблемы (обратная связь от владельца продукта)

1. Нет закруглённых углов на карточках
2. Нет типов задач (Фича, Баг, Техдолг)
3. Текст накладывается на тёмные панели
4. Нет фото исполнителей
5. Дизайн не соответствует макету

## Выполнено

### Изменения в `frontend/src/routes/tasks/+page.svelte`

1. **Task Interface**
   - Добавлен `task_type: 'feature' | 'bug' | 'tech_debt' | 'improvement' | 'task'`

2. **Константы**
   - `taskTypeLabels` - метки типов с цветами и иконками
   - `priorityBorderColors` - цвета левой границы по приоритету

3. **Функции**
   - `getEmployeePhoto(id)` - получение фото сотрудника

4. **Kanban карточки (полный редизайн)**
   - `rounded-lg` - закруглённые углы
   - `hover:-translate-y-0.5 hover:shadow-lg` - hover эффекты
   - `border-l-[3px]` - цветная граница по приоритету
   - Бейдж типа задачи с иконкой
   - Формат ID: EKF-XXXX
   - Фото исполнителя или инициалы в градиентном круге

5. **Модальное окно**
   - Добавлен селектор типа задачи
   - Grid изменён на 5 колонок
