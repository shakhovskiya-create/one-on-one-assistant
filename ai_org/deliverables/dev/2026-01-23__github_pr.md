## Функционал

### Backend
- `SearchPullRequestsForTaskID(owner, repo, taskID, limit)` - поиск PR по ID задачи
- Фильтрация по title и body PR на паттерн `(task|#|EKF-)taskID`

### Frontend
- Загрузка PR параллельно с коммитами
- Секция "Связанные Pull Requests" в модальном окне задачи
- Отображение: номер, статус, название, автор, дата
- Цветовая индикация:
  - Зелёный: Open
  - Фиолетовый: Merged
  - Красный: Closed

## API
- `GET /api/github/tasks/:id/pulls?owner=...&repo=...&limit=...`
